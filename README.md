# Минималистичный видеоплеер

## Описание

Приложение для локального просмотра видео с кастомным плеером.

## Запуск

1. npm install
2. npm run dev

## Функции

- Загрузка видео
- Play/Pause
- Прогресс-бар
- Время и длительность

## Структура

- src/video-player.tsx — компонент плеера
- src/App.tsx — точка входа
